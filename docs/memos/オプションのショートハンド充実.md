## 改善内容

### playコマンドで履歴を残さないオプションを追加
- `sine-mml play` コマンドに `--no-history` オプションを追加したいです。
- このオプションを指定すると、MMLの再生時に履歴に保存されなくなります。
- 例えば、以下のように使用します
```bash
sine-mml play "T120 L4 O4 CDEFGAB" --no-history
```
- これにより、一時的なMMLの再生やテスト演奏などで履歴を残したくない場合に便利です。
- ファイル読み取りや他のオプションと組み合わせて使用できるようにします。

### オプションのショートフォーム対応
- `--no-history` オプションにショートフォーム `-nh` を追加
- 例えば、以下のように使用できます
```bash
sine-mml play "T120 L4 O4 CDEFGAB" -nh
```
- 他のオプションと同様にショートフォームをサポートし、コマンドの入力を簡略化します。


### midi-outオプションとの併用確認
- `--midi-out` オプションと `--no-history` オプションの併用が可能であることを確認します。
- 例えば、以下のように使用できます
```bash
sine-mml play "T120 L4 O4 CDEFGAB" --midi-out 0 --no-history
```
- これにより、MIDIストリーミングを行いながら履歴を残さない設定が可能になります。

### midi-outオプションと--loop-playオプションとの併用確認
- `--midi-out` オプションと `--loop-play` オプションの併用が可能であることを確認します。
- 例えば、以下のように使用できます
```bash
sine-mml play "T120 L4 O4 CDEFGAB" --midi-out 0 --loop-play
```
- これにより、MIDIストリーミングを行いながらループ再生が可能になります。
