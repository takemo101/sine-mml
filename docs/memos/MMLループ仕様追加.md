## 仕様追加

### ループ仕様
MMLに対象の範囲をループさせる機能を追加したいです。
以下のような仕様がいいと思っています。

ＭＭＬ編集時、[]で囲うとループ設定になります。
[]の後に数値を付ければ、指定回数ループ。
数値がない場合はループしません（--loop-playなしに無限ループさせたくない）
ループ中に:を付ければループ脱出ポイントになります。
ループ回数残数が0のときは、:に来たらループを抜けます。


### 小文字記述
MMLのコマンドを小文字で記述できるようにしたいです。
例えば、CDEFGABやR、O5、L8、T140、V10などを
cdefgabやr、o5、l8、t140、v10のように小文字で書けるようにします。


### --noteオプション
コマンドに--noteオプションを追加し、メロディー履歴に任意のメモを付与できるようにしたいです。
例えば、以下のように使用します。
```
sine-mml play "T120 L4 O4 CDEFGAB" --note "My first melody"
```
この場合、履歴に「My first melody」というメモが保存されます。
履歴一覧表示時にメモも一緒に表示されるようにします。
これは --loop-play オプションで併用可能とします。

### 履歴クリア
履歴を全てクリアするコマンドを追加したいです。
例えば、以下のように使用します。
```
sine-mml clear-history
```
このコマンドを実行すると、保存されている全ての履歴が削除されます。
確認のプロンプトを表示し、誤操作を防止します。
