## 追加仕様

### ファイル読み取り仕様
現在コマンドでMMLを直接指定していますが、ファイルでのMMLの指定をしたいと考えています。
ファイルは会議用やタグを含みますが、その辺はMMLの演奏に影響が出ないようにしたいです。
ファイルで読み取るモチベーションとしては、長いMMLを小説や構成によって会議をタブなどで切り分けたいというものです。
対応するファイルの拡張子は.mmlを指定したいと思っています。

### ベロシティ指定
MMLでベロシティ（音の強弱）を指定できるようににしたいです。
例えば、Vコマンドでベロシティを指定できるようにします。
V0からV127までの範囲で指定でき、デフォルトはV100とします。
ベロシティ指定により、演奏される音の強弱が変化します。
相対的な音量調整も可能にしたいです。
例えば、V+10やV-10のように指定すると、現在のベロシティから相対的に増減します。
V+やV-のように指定した場合は、デフォルトの増減値（例えば10）を使用します。

### ループのネスト対応
ループ仕様において、ループのネスト（入れ子）に対応したいです。
例えば、以下のようなMMLをサポートします。
```
[ CDE [ FGAB ]2 ]3
```
この例では、内側のループ（FGAB）が2回繰り返され、その後に外側のループ（CDEと内側のループ全体）が3回繰り返されます。
ループのネストにより、より複雑なパターンを簡潔に表現できるようになります。
